# Allego Custom Skills

A collection of specialized AI skills for the Allego codebase.

## 🛠 Installation

1. Clone this repository to your local machine.
2. Run the installer script:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
3. Follow the prompts to select specific skills or install all of them.

## 📂 Included Skills

- **feature-context-generator**: Generates functional documentation and feature contexts from the Allego codebase.

## 🤝 Adding New Skills
To add a new skill to this repo:
1. Create a new directory.
2. Add a `SKILL.md` file (following the Superpowers format).
3. The `install.sh` script will automatically detect and offer it for installation.
